## Embedded System Lab project 2020
A Flappy Bird game implemented on STM32CubeIDE, built with C.

![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)

### Requirement
- ST NUCLEO-F4 Nucleo Development Board
- STM32CubeIDE
- Tera Term
